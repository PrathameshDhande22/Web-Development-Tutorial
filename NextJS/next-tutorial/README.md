# NextJS Tutorial

This folder contains my Next.js tutorial practice project and the topics covered while learning Next.js.

## Tutorial References

1. YouTube Tutorial: https://www.youtube.com/watch?v=k7o9R6eaSes
2. Official Documentation: https://nextjs.org/docs


## Topics Covered

1. Next.js introduction and App Router basics.
2. Creating pages with `page.tsx` and shared layouts with `layout.tsx`.
3. Nested routes and route groups.
4. Dynamic routes.
5. Catch-all and optional catch-all routes.
6. Special files like `loading.tsx`, `error.tsx`, `global-error.tsx`, and `not-found.tsx`.
7. Nested layouts for specific route sections.
8. Route handlers using `GET` and `POST`.
9. Working with route params, query params, headers, and cookies.
10. Server components and client components.
11. Server-side and client-side utility separation.
12. Data fetching in server components.
13. Caching with `use cache`, cache lifetime, and `no-store` fetching.
14. Server actions for form handling and mutations.
15. Redirecting and revalidating paths after mutations.
16. Database integration with Prisma and SQLite.
17. Product CRUD examples with Prisma.
18. Parallel routes in the dashboard example.
19. Intercepted routes and modal routing in the feed example.
20. Reusable components like cards, modal, header, footer, provider, and reviews.
21. Environment variables in Next.js.
22. Extra features like `sitemap.ts` and HTTP route testing.
