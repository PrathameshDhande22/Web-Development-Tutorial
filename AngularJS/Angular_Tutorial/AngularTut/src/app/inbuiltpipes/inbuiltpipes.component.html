Pipes are used to format the data or transform it before displaying it into the component.

1. UpperCasePipe
original => {{username}}
After Applying Pipe => {{username | uppercase}}
2. DataPipe
original => {{dateofbirth}}
After Applying Pipe => {{dateofbirth | date: "dd / LLLL / yyyy"}}
3. PercentPipe
original => {{percentage}}
After Applying Pipe => {{percentage | percent:"2.2" }}
4. Currency Pipe
original => {{money}}
After Applying Pipe => {{money | currency:"INR":true }}
