# NGRX Tutorial

NgRx angular State management Library tutorial. These tutorial will cover all the basics topics from the NgRx like
- Create Global Store
- Effects
- Actions, Reducers & Selectors
- Signal Based Local State Management

In these tutorial we have implemented the basic
- Counter with Increment and decrement using the Global Store
- Counter with Increment and decrement using the Signal Based Store
- Book Collection using the Effects and Global Store. 

## Book Collection Project with NgRx Tutorial

- Load the Books using the Effects and Some Actions.
- Add the books to the Book Store using the Effects.
- Add the books to the user collection.
- Show the books only where the user had added in his collection. 
- Remove the books from teh collection. 

### Detailed Step by Step Implementation: 
1. Load all the books
2. Create the Add and Remove Books from the Book Store
3. Add New Books
4. Remove Books
5. Add to User Collection
